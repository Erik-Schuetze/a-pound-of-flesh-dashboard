What is this?
This is a Dashboard for 'A Pound of Flesh' - a module for the Mothership RPG
It is meant as a background on a display somewhere.
How does this work?
You can add or remove Players by using the + and - button in the footer menu. This data is saved to your browser.
Edit player names and Roles simply by clicking on them and typing the info in.
To make a backup simply export the data and import it when you need it with the cloud shaped buttons.
the main Display on the right side can be used to display maps as a background for your APOF game.
